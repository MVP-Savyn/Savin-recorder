# Savin-recorder
This is a screen recorder for Hyprland, but… the interesting point is the fact that it uses (optionally) cloud services of your choice for various purposes; for example, bypassing Discord’s file size limit restriction. (Still on developing)
<br> <br>
Este es un grabador de pantalla para Hyprland, pero… lo interesante es que utiliza (de forma opcional) servicios en la nube a elegir para distintos propósitos; por ejemplo, saltarse la restricción de límite de tamaño en Discord. (Todavía está en desarrollo)
<br> <br>
La primera versión está terminada, los archivos se pueden descargar e instalar ejecutando el comando ./install.sh
Seguiré trabajando en esta pequeña aplicación y en la explicación de cómo usarlo, por ahora sólo quiero tenerlo guardado.
